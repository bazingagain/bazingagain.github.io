什么是SpringBoot？

SpringBoot 是Spring领域中的一个全新的项目，它提供了快速便捷的方式开发Spring项目，简化了传统的Spring配置项。它能够快速的构建独立的、工业化级别的Spring应用并运行。

SpringBoot主要提供了如下特性：

  • Spring Boot Starter: 对常用依赖分组进行整合，合并到一个依赖中，简化依赖配置
  • 自动配置：SPring Boot 自动推测所需要的bean并配置它们
  • 命令行接口：CLI，提供SpringBoot CLI支持Groovy
  • Actuator：为SpringBoot提供一些管理特性

什么是Spring Boot Starter

Spring Boot Starter 是将常见的依赖组合聚合在一起而构成的一项依赖，如当我们要编写SpringMVC应用时，传统的做法是添加core、context、mvc等依赖，而利用starter则只需添加pring-boot-starter-web：

pring-boot-starter-web包含了pring-boot-starter、pring-web、pring-webmvc、jackson-databind等依赖，这样我们只需添加一项就可以导入所需的多个依赖，简化了pom.xml的配置。

使用Spring Boot Actuator

/beans：列出运行应用的所有bean
/info: 应用的特定信息
/health: 应用的健康状况
