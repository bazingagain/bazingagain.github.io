什么是MongoDB？

MongoDB是一款NoSQL数据库，其使用C++编写，是一个基于分布式存储的开源数据库系统，旨在为应用提供可扩展的高性能数据存储解决方案。

MongoDB的特点是什么？

  • MongoDB是面向文档存储的数据库
  • 其支持多种查询方式，如类SQL方式、json方式，查询方式简单易懂
  • 支持map/reduce操作进行数据批量处理和聚合
  • 内置的GridFS提供存放大量小文件的能力

MongoDB中的一些基本概念

SQL概念 MongoDB概念 说明
database database 数据库
table collection 集合
row document 文档
column field 字段
index index 索引
table join 不支持（MongoDB使用内嵌方式进行关联） 表连接
primary key primary key 主键，MongoDB自带_id主键

MongoDB中，document是一组 key/value 对（BSON），每个文档并不需要设置相同的字段，相同字段也不需要设置相同的类型
