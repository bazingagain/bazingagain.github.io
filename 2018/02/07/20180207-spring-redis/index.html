Spring学习之整合Redis

序言

Redis是一种非常流行的key-value数据库，支持list、set、string、zset、hash等数据类型，支持数据的持久化、数据的备份等。而当使用Spring框架构建项目时，使用redis可以作为缓存支持提供快速访问数据的能力。

Spring整合Reids

Spring整合redis使用的是Spring Data Redis实现的，其提供多种连接工厂和多种模板实现来完成Redis的数据存取功能。

Spring Data Redis 提供的四种连接工厂：

  • JedisConnectionFactory
  • JredisConnectionFactory
  • LettuceConnectionFactory
  • SrpConnectionFactory

Spring Data Redis 提供的两种模板以方便数据访问：

  • RedisTemplate
  • StringRedisTemplate

RedisTemplate可极大简化redis的访问，支持各种类型的key和value，StringRedisTemplate扩展了RedisTemplate,只关注String类型。

值得注意的是：当需要使用Redis的k-v存储时，k和v会使用Redis的序列化器进行序列化。Spring Data Redis提供了多种序列化器：

  • GenericToStringSerializer：使用Spring转换服务进行序列化
  • JacksonJsonRedisSerializer: 使用Jackson1将对象序列化为JSON
  • Jackson2JsonRedisSerializer: 使用Jackson2将对象序列化为JSON
  • JdkSerializationRedisSerializer: 使用Java序列化
  • OxmSerializer: 使用Spring O/X映射的编排器和解牌实现序列化，用于XML序列化
  • StringRedisSerializer: 序列化String类型的key和value

其中，RedisTemplate使用JdkSerializationRedisSerializer进行Java序列化，而StringRedisTemplate则使用StringRedisSerializer进行序列化。
