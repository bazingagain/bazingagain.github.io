简介

Apache Hadoop 是一个分布式计算与处理框架，它能够使用计算机集群对大规模的数据进行分布式计算和处理。

Hadoop 主要包括以下四个模型：

  • Hadoop Common : 支持Hadoop模块的通用工具集
  • Hadoop Distributed File System (HDFS) : 分布式文件系统，其提供了对硬件错误的检测和容错处理，提供高吞吐量的数据访问，一个HDFS实例能够支撑数以千万计的文件。
  • Hadoop YARN ： 任务调度与集群资源管理框架
  • Hadoop MapReduce ： 基于YARN的大型数据集并行处理系统

在安装好之后，进行Hadoop文件的配置

1. 配置SSH：

  • 1.打开mac远程登录： system preferences -> share 勾选 Remote Login
  • 2.Terminal中使用命令生成rsa公钥： ssh-keyagent -t
  • 3.设置免登：cat ~/.ssh/id_rsa.pub > ~/.ssh/authorized_keys
  • 4.验证SSH是否配置成功: ssh localhost
