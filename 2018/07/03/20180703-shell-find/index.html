Shell学习之find命令的使用

发表于 2018-07-03 | 分类于 Shell | | 阅读次数 | 访问次数 次
字数统计 986 | 阅读时长 4

find命令是linux shell脚本下使用频率非常高的命令，其功能非常强大，对于我们在linux下对文件进行快速查找来说是一种非常高效的方式。

find的基本用法

1.打印当前文件下的所有文件和目录

2.按文件名匹配( -name)

3.按路径匹配( -path)

4.按正则表达式匹配( -regex)

5.按文件类型搜索 ( -type [fldcbsp])

Linux中将一切视为文件，只是文件的类型不同，主要的文件类型对应type的参数如下：

文件类型 参数
普通文件 f
符号链接 l
目录 d
字符设备 c
块设备 b
套接字 s
Fifo p

6.否定参数(!)

使用 ! 否定参数，可以查找条件匹配的补集

find的高级用法

1.按目录深度查找（ -maxdepth, -mindepth）

2.按文件时间进行查找( -atime, -mtime, -ctime, -newer)

  • -atime, -amin : （访问时间）—— 用户最近一次访问文件的时间
  • -mtime, -mmin : （修改时间）—— 文件内容最后被修改的时间
  • -ctime, -cmin : （变化时间）—— 文件元数据（如权限、所有权）最后一次改变的时间

atime, mtime 以及 ctime 是以 天 为单位计算时间的:

amin, mmin 以及 cmin 是以 分钟 为单位计算时间的:

-newer 是一种比较少见的方式，其是找出比参考文件修改时间更长的所有文件：

3.按文件大小查找( -size)

按文件大小查找文件，其参数有 k, b, c, w, M, G

  • k : kB
  • b : 块(512B)
  • c : 字节(B)
  • w : 字(2B)
  • M : MB
  • G : GB

4.匹配删除( -delete)

5.根据文件权限或用户名查找( -perm, -user)

6.跳过某个目录查找( -prune)
