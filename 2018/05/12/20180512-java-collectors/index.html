Stream收集器的概念

Java中，对于集合的操作，可以将其转换为流，组装成流水线进行数据的处理，最后，我们要得到一个确切的结果，在Java中，对流的处理的最后步骤一般是使用收集器进行流的聚合整理得到最终的结果。

为了方便收集器的使用，Java提供了Collector类，其包含众多常用的收集器工厂方法。按收集器的功能划分主要可以分为以下几类：

  • 将流元素归约和汇总为一个值
  • 元素分组
  • 元素分区
