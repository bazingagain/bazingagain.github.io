什么是Java Stream？

Java Stream 是Java8新添加的API，在java.util.stream下，其提供透明的声明式的方式对数据集合进行处理，使用Java Stream对数据进行处理就像构造一条数据流水线一样，通过输入、过滤、映射以及整合等处理步骤最终得到我们想要的结果。流与集合不同，集合主要是针对数据和结构的概论抽象，而流则是对数据计算的抽象。

流(stream)的数据是按需加载的，你只需提取你所关心的数据，另外，流的处理过程类似于流水线，与迭代器类似，流只能遍历一次，如果把人的一生看做是一个流处理，就像人生只有一次一样，虽然我们在空间的角度上可以到达同一个地方，但是其实按照时空维度来讲，同一个地方我们是不可能出现两次的，因为时空是不断向前流动的。同样，Java中，一个流的处理也是数据不停向前流动的过程，不能回溯。

Stream的使用

对于Stream的使用，Java提供了许多函数供我们使用，由于Stream将数据从外部迭代转化为了内部迭代，在使用上，简化了集合的数据操作，更利于并行化处理数据。
