K-Means 算法介绍

k-Means算法是一种简单的聚类算法，对于n个点，将其聚类到k个簇中。
引用维基百科k-Means的算法描述：

对于输入的观察集(x1, x2…,xn)，其中每个观测都是一个 d-维实向量，k-平均聚类要把这n个观测划分到k个集合中(k≤n),使得组内平方和 （WCSS within-cluster sum of squares）最小。换句话说，它的目标是找到使得下式满足的聚类 Si，

K-Means算法的过程

k-Means 算法一般采取如下步骤进行数据聚类：

  • (1)随机创建k个点作为质心
  • (2)对每个点，计算其到每个质心的距离，将其分配到距离其最近的质心（簇）
  • (3)对每个簇，计算簇中所有点的均值并将均值作为质心
  • (4)重复2~3步，知道新的质心与原质心相等结束

K-Means算法的优缺点

优点：
算法简单，容易实现
时间复杂度接近于线性，适合大数据处理

缺点：
k-Means算法收敛到局部最小，而非全局最小（即可能并非最好结果）
k是事先给定的，而如何指定k则没有标准

而该方法存在一个问题是，算法会收敛于局部最小，得到的效果并不是全局最优，为了解决该问题，提出了 二分k-Means算法。

二分k-Means算法

二分k-Means算法的过程如下：

  • (1)将所有点看成一个簇
  • (2)当簇的数目小于k时，对每个簇计算其总误差，在给定的簇上进行k-均值聚类(k=2)，计算该簇被一分为二之后的总误差，选择误差最小的那个簇进行划分
